O que é HTML?

O significado de HTML é linguagem de marcação de hipertexto, significa que ele é o componente básico da estrutura de um site. Ele serve para mostrar ao navegador como se "comportar" ao lidar com textos, imagens e elementos usados na linguagem em que foi programado determinado site. Apesar do seu nome ter esse significado, HTML não é uma linguagem de programação pois ele não tem funcionalidades dinâmicas então ele trabalha em conjunto com a linguagem escolhida pelo programador para armazenar e exibir as informações contidas nos sites.

Motivos para aprender HTML antes de programar

Primeiramente, para iniciantes é uma ótima maneira de vizualizar como é programar, com funções faceis de entender e explorar, você consegue de forma rápida ver um resultado mais simples direto no seu navegador. Ajuda a treinar os primeiros conceitos e além de tudo, é compatível com todos os navegadores.

Na prática

Os passos iniciais para entender HTML é aprender as tags e o que elas fazem, respeitando a ordem de cada uma e para te ajudar a ver o que está fazendo, pode ir salvando tudo que escreveu no código e visualizar em tempo real no seu navegador como abaixo:

Toda tag deve ser aberta e fechada para que o comando funcione com < > fechando com < / tag que você esta usando >. Veja alguns exemplos de comandos muito úteis:

  • html : Tipo do documento no início do código

  • head : Cabeçalho da página

  • title : Quando usada como tag dentro da sessão head serve para inserir o título da página. Quando usada como parâmetro em uma função serve para inserir textos informativos quando necessário

  • body : tudo que irá no "corpo" do seu site, colocado depois do < head > e por ultimo antes do < /html >

  • i : para escrita em itálico

  • strong : para escrita em negrito

  • style="color:" cor (em inglês) " : adicionando a cor que você quer deixar o texto

  • h1 até h6 : para letras maiores e titulos capitulares

  • hr /: quebra de linha

  • ul : para lista sem numeração de linha

  • ol : para lista numerada

  • li : para os tópicos da lista

  • p : para parágrafo

  • blockquote : Para citações

  • img src=" " : colocando o link imagem entre aspas, a imagem estará no site

  • input type="text" : para adicionar campo de preenchimento colocando / antes de fechar pois não precisa fecha-la separadamente

  • input type="number" : para adicionar campo de preenchimento numérico

  • input type="color" : para campo de cor selecionável

  • mark : Para marcador de texto

  • s : para texto riscado

  • small : para texto menor

  • sub : Para texto subscrito

  • sup : Para texto sobrescrito

  • big : Para texto maior

Você encontra essas e muitas outras funcionalidades aqui.
